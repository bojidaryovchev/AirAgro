## 4. СТРАТЕГИЯ ЗА ВЪТРЕШНО СВЪРЗВАНЕ

### Принципи:

1. **Хъб-и-спици модел:** Главната услугова страница линква към всички crop-specific и регионални статии
2. **Агрономическият контент** линква към crop-specific статии и услуги
3. **Сравнителните статии** линкват към услуги и ценова страница
4. **Образователният контент** линква към сравнения и услуги
5. **Регионалните страници** линкват към услуги, цени и crop-specific съдържание

### Примерна мрежа:

```
Услуги (хъб)
  → Цена
  → Всички crop-specific
  → Регионални

Crop-specific (пшеница)
  → Кога се пръска пшеница
  → Борба с плевели пшеница
  → Фунгицидно пръскане пшеница
  → Листно торене пшеница
  → Услуги
  → Цена

Агрономия (кога се пръска пшеница)
  → Пръскане на пшеница с дрон
  → Фунгицидно пръскане
  → Дрон или трактор
  → Услуги

Сравнения (дрон или трактор)
  → Предимства на дрон
  → Колко декара пръска дрон
  → Услуги
  → Цена

Регионални (Пловдив)
  → Услуги
  → Цена
  → Crop-specific за региона
```

## 5. SEO ОПТИМИЗАЦИЯ – ПРАВИЛА ЗА ВСЯКА СТАТИЯ

1. **Title tag:** Основна ключова дума + допълнение (до 60 символа)
2. **Meta description:** Включва ключова дума + CTA (до 155 символа)
3. **H1:** Едно H1 = заглавието на статията
4. **H2/H3:** Включват семантични вариации на ключовата дума
5. **Въведение:** Ключовата дума в първите 100 думи
6. **Заключение:** CTA с линк за контакт
7. **Вътрешни линкове:** Минимум 3-5 към други статии
8. **Параграфи:** 2-4 изречения, лесни за четене
9. **Списъци и таблици:** Когато е уместно, за по-добро потребителско изживяване
